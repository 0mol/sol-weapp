## 开发

开发者可以通过 git clone 下载项目文件，进行开发。
使用微信 web 开发者工具 打开 example 目录预览效果

```bash
git clone https://github.com/sunniejs/sol-weapp.git
```

## 开发之前

- 全局安装 gulp-cli

```bash
npm install --global gulp-cli
```

- 根目录执行，安装必要模块

```bash
npm install
// or
npm i
```

- 根目录执行，本地开发

```bash
npm run start
```

- 根目录执行，打包线上

```bash
// 根目录执行
npm run build
```
